关键发现:
1. TokenFormer 核心架构 (统一 Token 流 + BFTS 注意力) 是提升关键，Avazu 上 +0.0015
2. Per-Field Tokenization 贡献微弱 (+0.0007)，但正向
3. SWA 非常重要: 禁用后 AUC 下降 -0.0039 (0.7463→0.7424)
4. Global Token 贡献微弱 (+0.0005)
5. MoE 贡献 +0.0010 (微弱)
6. Inter-layer Residuals 非常重要: 禁用后 AUC 下降 -0.0120 (0.7463→0.7343)
7. MoE+Aux+Contrast 联合 贡献 +0.0010 (微弱，M3f 仅 P0+P1+P2 与 M3e 无 MoE 持平)
